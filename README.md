## ðŸš€ Getting Started

### Prerequisites

- Node.js > 18
- MongoDB

### Installation

1. **Clone the repository**
2. **Install Dependencies**

   ```bash
   # Backend
   cd server
   npm install

   # Client
   cd client
   npm install
   ```

3. **Run Development Servers**

   ```bash
   # Backend
   cd server
   npm run dev

   # Client
   cd client
   npm run dev
   ```
